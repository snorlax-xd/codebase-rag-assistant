## Project Identity
- **Project Name:** CodeMind AI — Codebase RAG Assistant
- **Purpose:** A full-stack AI-powered platform where users paste any GitHub repo URL, the system clones + indexes it, and users can chat with the codebase, search semantically, visualize architecture, and browse files.
- **Live URLs:**
  - Frontend: https://codebase-rag-assistant.vercel.app
  - Backend: https://codebase-rag-assistant-production.up.railway.app
  - Backend Swagger: https://codebase-rag-assistant-production.up.railway.app/docs

---

## Tech Stack

### Frontend
- **Framework:** Next.js 16.2.6 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS with custom design tokens
- **Animations:** Framer Motion
- **Code Editor:** Monaco Editor (via `@/components/features/CodeEditor`)
- **Architecture Graph:** ReactFlow (`@xyflow/react`)
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Root Directory on Vercel:** `frontend/codemind-ai`
- **Env Variable:** `NEXT_PUBLIC_API_URL=https://codebase-rag-assistant-production.up.railway.app`

### Backend
- **Framework:** FastAPI (Python 3.11)
- **LLM:** Gemini 1.5 Flash (via direct REST API — NOT the SDK)
- **Embeddings:** Gemini Embedding 001 (via direct REST API, v1beta endpoint)
- **Vector DB:** Qdrant (running as Docker service on Railway)
- **Code Parsing:** tree-sitter (Python only; other languages use raw content fallback)
- **Git Cloning:** GitPython
- **Deployment:** Railway (Docker)
- **Env Variable:** `GEMINI_API_KEY=<actual key>`

### Key Backend Facts
- Embedding model: `gemini-embedding-001` via `https://generativelanguage.googleapis.com/v1beta/models/gemini-embedding-001:embedContent`
- Vector dimensions: **3072** (gemini-embedding-001 output size)
- LLM model: `gemini-1.5-flash` via `https://generativelanguage.googleapis.com/v1/models/gemini-1.5-flash:generateContent`
- Qdrant collection name: `codebase_chunks`
- Rate limit handling: 0.7s proactive delay + retry on 429/503 (up to 7 retries)
- Indexing is **background task** — returns immediately with `indexing_started`
- CORS allows: `localhost:3000` + `https://codebase-rag-assistant.vercel.app` + `https://*.vercel.app`

## LocalStorage Keys (Frontend State)
| Key | Purpose |
|-----|---------|
| `codemind_repos` | Array of added repositories (persisted across sessions) |
| `codemind_active_repo` | Currently selected repo name for chat/search/architecture |
| `codemind_chat_history` | Array of past chat sessions |
| `codemind_current_session_id` | Active session ID being written to |
| `codemind_prefill_query` | Query prefilled from semantic search "Ask AI" button |

## Backend API Endpoints
| Method | Endpoint | Purpose |
|--------|----------|---------|
| POST | `/clone-repo?repo_url=` | Clone a GitHub repo |
| GET | `/scan-repo?repo_name=` | Get file tree of a cloned repo |
| POST | `/create-collection` | Create Qdrant collection |
| DELETE | `/delete-collection` | Delete Qdrant collection |
| POST | `/index-repo?repo_name=` | Start background indexing |
| GET | `/search?query=` | Semantic search |
| GET | `/ask?query=` | RAG question answering |
| GET | `/collection-info` | Vector count in Qdrant |

13. Indexing crashed entire job on single chunk failure — wrapped each chunk in try/except/continue

## Important Rules for Future Sessions
1. **Always stress test before writing code** — check for TypeScript errors, missing imports, wrong variant names, key mismatches
2. **Never use `Badge variant="optimal"` or `variant="critical"`** — these don't exist; use inline span with conditional classes
3. **Never use `Button variant="neon"`** — doesn't exist; use `variant="secondary"`
4. **`repositories/page.tsx` must be explicitly `git add`ed** — it was never tracked
5. **Gemini SDK (`google-generativeai` / `google-genai`) should NOT be used** — use direct HTTP REST calls only
6. **Vector dimensions must be 3072** for `gemini-embedding-001`
7. **Indexing is always async/background** — never await it synchronously
8. **`codemind_active_repo` in localStorage** is the source of truth for which repo is active
9. **`app/page.tsx` is a redirect** — not a layout, not a landing page
10. **Vercel builds from GitHub** — local changes must be `git add` + `git commit` + `git push` to take effect
